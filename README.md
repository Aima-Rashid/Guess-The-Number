# Guess The Number Game

This project is a simple number guessing game developed using HTML, CSS, and JavaScript.

## Game Rules
- The computer randomly selects a number.
- In Level 1, the number is between 1 and 10.
- The user has a maximum of 3 attempts to guess the number.
- If the guessed number is lower than the actual number, a "Too Low!" message is shown.
- If the guessed number is higher than the actual number, a "Too High!" message is shown.
- Invalid inputs (empty, non-numeric, or out-of-range values) are ignored and do not count as attempts.
- If the user guesses correctly within 3 attempts, the game moves to Level 2.
- In Level 2, the number range becomes 1 to 25.
- The user again has only 3 attempts.
- If the user fails to guess the number within the allowed attempts at any level, the game ends.
- A restart button appears after game over or after winning the game.

## Technologies Used
- HTML for structure
- CSS for styling
- JavaScript for game logic

## Features
- Input validation
- Attempt limitation
- Level progression
- Restart functionality
- Clean and user-friendly interface

This project is suitable for beginners learning JavaScript and basic web development concepts.
